CSS Box Model

Layers of Box Model

  • Margin : Margin was most outer layer of box-model.
  • Border : Border was the second layer after margin.
  • Padding: Padding was the third layer after border.
  • Content box : Content box the inner layer in which our content we define this content box we give width and height.
