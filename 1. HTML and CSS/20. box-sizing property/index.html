Box sizing property

box-sizing property was important was used to set the sizing of our content box means height and width of our html element.

box-sizing properties have two values

content-box property

In this content-box property value it was only calculated width and height of our content box if we give padding and border than it was calculated extra in our height and width and increase both.

Example of content-box value

We use `content-box` value in this div box-sizing property.

So above height and width was actually we give height = 200px and width = 400px

We give padding and border in four direction so total padding was padding = 20 + 20 + 20 + 20 = 80px and border = 10 + 10 + 10 +10 = 40px

border-box

In this border-box value the border and padding was also adjust in height and width also not added in extra spaces We have to always give every html element box-sizing : border-box

Example of border-box

We give this div `border-box` value of box-sizing property

In above div we give height = 200px and width = 500px after the property box-sizing:border-box and giving padding = 20px in all 4 direction and border: 10px also in 4 direction so this was like this height = 200 - 20 + 20 + 10 + 10 = 200 - 40 + 20 = 200 - 60 = 140 and width = 050 - 20 + 20 + 10 + 10 = 500 - 40 + 20 = 500 - 60 = 440 so total height and width was 140 x 440
