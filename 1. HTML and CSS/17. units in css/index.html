Units in CSS

Pixels Unit

Pixel unit was absolute length unit

Example of pixel unit

We give this width and height in 300 px "pixels units"

Percentage unit "%"

Percentage unit was relative length unit so it was relative to parent length and if we want to use relative length we have to give absolute unit to our parent element than percentage unit work or than not.

Examples of Percentage unit

This section element we give height and width 600px
We give this div element height and width in percentage height was 40 % and width was 100% which was calculated from parent's width and height.

em unit

em unit was also relative unit which depends on font-size of our parent element the default font-size was 16px.

Example of em unit

Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolor quidem excepturi iusto fugiat officiis, eligendi quae quibusdam animi cupiditate quos? Lorem ipsum dolor sit amet consectetur adipisicing elit. Rerum, praesentium!

In above em unit example the parent element paragraph has font-size 1 em = 16px and the text which are in red was span tag which was children of p tag that font-size was 2em which was 2 * 16 = 32px

rem unit

This rem unit was also relative length but rem unit relative to root element which was html tag it means when we are calculate font-size of any html element with root element { html tag} when we are using rem unit.

Example of rem unit

Lorem, ipsum dolor sit amet consectetur adipisicing elit.

In above example we used rem unit which relative to root element aka html element and we know that default font-size was 16px and we applied 3rem so that was 3 * 16 = 48 px

Most of html element have default font-size was 16px execept button element etc.

vw unit

vw was also relative length which stand for Viewport width. This vw unit was relative to viewport of our device or the body area { Browser screen where our website visible } of our html element.

vw : this unit was relative to 1% of the width of the viewport.

Example of vw unit

1%
50 % of the screen or viewport
100 % of the screen or viewport

vh unit

vh unit was relative unit which was similar to vw unit but here vh unit was relative to viewport height it means our height of html element was calculated from device viewport or screen of browser where we used web-application.

vh : this unit was relative to 1% of the height of the viewport.

Example of vh unit

1%
50%
100%
