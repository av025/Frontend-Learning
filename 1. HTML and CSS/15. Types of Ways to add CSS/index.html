There was 3 ways to add CSS

  • Inline Styling
  • Internal Styling
  • External Styling

External Styling

In External Styling we create external css file with .css extension in which we write the css properties externally not in html document and we have to link the external css with our html document which was most important to link our css with html document with the help of link tag in html.
